# app1-starsky-notes

## 项目简介

`app1-starsky-notes` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
这些页面通常以笔记、工具界面或简单应用的形式存在，彼此之间没有强依赖关系。  
本仓库主要用作静态页面集合的存储与展示，不针对任何特定域名或网站。

## 目录说明

```
.
├── index.html          # 仓库主页，列出所有归档页面的链接
├── pages/              # 存放各个独立 HTML 页面
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
├── assets/             # 公共资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：每个 HTML 文件代表一个独立的归档页面，可直接在浏览器中打开。
- `assets/`：存放多个页面可能共用的样式、脚本或图片资源。

## 页面归档说明

- 所有归档页面均为独立的 HTML 文件，不依赖后端服务。
- 每个页面内容以笔记或实用小工具为主，风格简洁，便于直接查看或离线使用。
- 归档页面会不定期更新或补充，不保证持续维护所有页面。

## 维护说明

- 本仓库由个人维护，用于整理和分享一些零散的 HTML 页面。
- 如需新增页面，请将 HTML 文件放入 `pages/` 目录，并在 `index.html` 中添加对应链接。
- 欢迎通过 Issues 或 Pull Requests 提出改进建议，但请注意本仓库不接收商业或营销内容。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 打开 `index.html` 查看页面列表。
3. 点击任意链接打开对应的独立页面。
